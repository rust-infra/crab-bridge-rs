Welcome to CrabBridge

Connect Codex CLI to DeepSeek and Kimi through a local bridge. API keys stay in your shell — the same DEEPSEEK_API_KEY / KIMI_API_KEY Codex already uses.

1. Environment

CrabBridge reads the same env vars as Codex. No keys are stored in the app unless you opt in via Settings.

Checking…

2. Configure Codex + Bridge

Writes ~/.codex/config.toml, model catalogs, and ~/.config/crabbridge/config.toml.

3. Start Bridge

Runs the local HTTP proxy Codex talks to. Codex sends your API key on each request — the bridge does not need its own copy at runtime.

Finish setup to see connection details.
